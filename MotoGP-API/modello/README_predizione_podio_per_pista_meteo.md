# README - predizione_podio_per_pista_meteo.py

Questo script calcola la probabilità di podio per piloti MotoGP in base a pista e condizioni meteo, utilizzando modelli di classificazione.

## Funzionalità
- Addestramento modello su dati storici
- Predizione podio per combinazioni pista/meteo
- Esportazione risultati e grafici

## Utilizzo
Esegui lo script:
```
python modello/predizione_podio_per_pista_meteo.py
```

## Output
- CSV con probabilità podio per pista/meteo
- Grafici metriche

## Dipendenze
- pandas
- scikit-learn
- matplotlib
- seaborn

## Note
Verifica che i valori di pista e meteo siano presenti nel dataset.
