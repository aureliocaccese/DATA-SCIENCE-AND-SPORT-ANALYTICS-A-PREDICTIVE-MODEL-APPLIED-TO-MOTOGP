# README - predizione_tempo_giro.py

Script per la predizione del tempo/gap sul giro per piloti MotoGP tramite modelli di regressione.

## Funzionalità
- Addestramento modelli di regressione
- Predizione tempo/gap per pilota/pista
- Esportazione risultati e grafici

## Utilizzo
Esegui lo script:
```
python modello/predizione_tempo_giro.py
```

## Output
- CSV con tempi/gap previsti
- Grafici errori e performance

## Dipendenze
- pandas
- scikit-learn
- matplotlib
- seaborn

## Note
Verifica la qualità dei dati prima dell'addestramento.
